# SillyTavern GPT Message Rail

A sleek navigation rail extension for SillyTavern that replaces standard message scrolling with an interactive, expanding side-panel.

## Features

- **Expanding Hover Panel**: Hovering the rail expands it into a fully readable list of messages with sleek glassmorphism styling.
- **Kinetic Swipe-to-Scroll**: Click and drag anywhere in the expanded rail to glide through your chat history with native-feeling, high-performance momentum scrolling.
- **Cross-Browser Styling**: Custom dark-themed scrollbars consistent across both Chrome and Firefox.
- **Instant Message Jumping**: Click any dot or message in the rail to instantly scroll your chat directly to that message.
